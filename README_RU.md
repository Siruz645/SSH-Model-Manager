# SSH Model Manager для ComfyUI

Кроссплатформенное десктопное приложение, написанное на Python и PyQt6, разработанное для управления, скачивания и загрузки моделей на удаленные серверы ComfyUI.

## Возможности

- **🚀 SFTP-загрузка (Drag & Drop):** Просто перетащите любой файл модели со своего компьютера в окно приложения для загрузки на удаленный сервер через оптимизированный SFTP.
- **🔗 Скачивание по ссылкам:** Вставьте ссылку на скачивание (с Civitai, HuggingFace и т. д.), и программа отдаст команду удаленному серверу скачать её напрямую.
- **🔑 Подстановка токенов:** Автоматически подставляет ваши API-токены HuggingFace или Civitai в запросы, чтобы вы могли скачивать приватные или закрытые модели без ввода ключей каждый раз.
- **📦 Пакеты моделей:** Группируйте несколько моделей в кастомные «Пакеты». Экспортируйте эти пакеты в файлы `.json` (например, если вы создали workflow, вы можете приложить этот json, чтобы другим было удобно скачать всё необходимое) или импортируйте пакет перетаскиванием для массового скачивания.
- **⚡ Фоновое сканирование:** Приложение автоматически сканирует папку `models` вашего сервера каждые 10 секунд и синхронизирует её с локальным каталогом, обнаруживая файлы `.safetensors`, `.pt` и `.bin`.
- **⏸️ Управление очередью:** Скачивание и загрузка нескольких файлов выполняются последовательно. Вы можете ставить задачи на паузу, возобновлять и отменять их. При отмене задачи с сервера автоматически удаляются частично скачанные файлы.
- **🧹 Удаление:** Удаляйте файлы с удаленного сервера для освобождения места, сохраняя при этом информацию о модели в вашей библиотеке для будущих загрузок.
- **📥 Добавление в библиотеку:** Вы можете вносить файлы, уже находящиеся на сервере, в свою локальную библиотеку (привязав к ним ссылки на скачивание) для быстрых повторных закачек в будущем.
- **🌍 Интерфейс:** Поддержка 3-х языков (английский, русский, китайский).

## Требования

- Python 3.9+

## Установка и использование

**Для пользователей Windows (самый простой способ):**
1. Склонируйте репозиторий или скачайте архив ZIP.
2. Дважды кликните по файлу `start.bat`.
   *(Он автоматически создаст виртуальное окружение, установит необходимые зависимости и запустит приложение).*

**Ручная установка (Linux/Mac/Продвинутый уровень):**
1. Склонируйте репозиторий:
   ```bash
   git clone https://github.com/Siruz645/SSH-Model-Manager.git
   cd SSHModelManager
   ```
2. Установите зависимости:
   ```bash
   pip install -r requirements.txt
   ```
3. Запустите приложение:
   ```bash
   python main.py
   ```

## Руководство пользователя

### 1. Подключение
На первой вкладке вы можете ввести данные для SSH-подключения к вашему серверу. Вы также можете быстро вставить команду SSH целиком (например, `ssh root@192.168.1.10 -p 22`), и поля заполнятся автоматически. Поле пароля можно оставить пустым; вы также можете указать путь к вашему SSH-ключу и ввести пароль от ключа, если требуется. Не забудьте ввести свои API-токены, если планируете скачивать приватные модели!

### 2. Загрузки
Вставляйте сюда ссылки для запуска удаленных загрузок или перетаскивайте файлы со своего рабочего стола для запуска SFTP-загрузки. Активные задачи можно поставить на паузу или отменить в любое время.

### 3. Библиотека
Просмотр всех скачанных или отсканированных моделей на вашем сервере.
- Отмечайте галочками модели и нажимайте **Создать пакет**, чтобы сгруппировать их.
- Нажмите правой кнопкой мыши (ПКМ) на любой пакет, чтобы **Экспортировать** его в файл `.json` или удалить.
- **Перетащите (Drag & Drop)** любой ранее экспортированный файл `.json` в окно программы, чтобы импортировать пакет и начать массовое скачивание всех его моделей.
- **Массовое добавление в очередь:** Выделите галочками файлы или целые пакеты в библиотеке и нажмите "Добавить выделенное в очередь" для их пакетного скачивания.

## Используемые технологии
- **PyQt6**: Для графического интерфейса.
- **Paramiko**: Для SSH-подключений и SFTP-передачи файлов.
- **SQLite**: Для легкого локального кэширования библиотеки моделей.
